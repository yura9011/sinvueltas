# Sin Vueltas - Website

Landing page para **Sin Vueltas**, rotisería y comida casera en Godoy Cruz, Mendoza.

## Sobre el negocio

- **Especialidad:** Pollo al spiedo, hamburguesas caseras, pizzas, empanadas
- **Ubicación:** Castellani 224, Godoy Cruz (frente al Club Talleres)
- **Horarios:** Mar-Dom 12-15hs | Vie-Sáb 20-23hs | Lunes cerrado
- **Contacto:** 261-263-9623

## Deploy

Hosteado en [Vercel](https://vercel.com).

```bash
# Clonar el repo
git clone https://github.com/yura9011/sinvueltas.git
cd sinvueltas

# Abrir localmente
# Solo abrir index.html en el navegador
```
